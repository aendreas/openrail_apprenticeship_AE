# CIF_catering_code
Up to two characters from the following:
* *C* - Buffet Service
* *F* - Restaurant Car available for First Class passengers
* *H* - Hot food available
* *M* - Meal included for First Class passengers
* *P* - Wheelchair only reservations
* *R* - Restaurant
* *T* - Trolley service
